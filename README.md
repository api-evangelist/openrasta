# OpenRastaThis is a repo for managing machine readable artifacts for the [OpenRasta APIs](http://openrasta.org/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [OpenRasta APIs](http://openrasta.org/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://openrasta.org/)- [Developer](http://openrasta.org/)- [GitHub](https://github.com/openrasta)The [APIs.json](https://github.com/api-evangelist/openrasta/blob/master/apis.json) index provides links to all the relevant links for the OpenRasta APIs, including the OpenAPI and Postman if they are present.
